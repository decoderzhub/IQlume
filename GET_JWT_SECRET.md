

# To get your Supabase JWT Secret:
# 1. Go to https://supabase.com/dashboard
# 2. Select your project (qvsjmfyedohwtzpuvilr)
# 3. Go to Settings > API
# 4. Look for 'JWT Secret' section
# 5. Copy the JWT Secret value
# 6. Add it to backend/.env as SUPABASE_JWT_SECRET=<your_secret>
#
# The JWT secret is used to verify user authentication tokens.
